markdown# Проект автоматизации тестирования с Allure-отчетами

Проект содержит UI-тесты, написанные на Python с использованием Pytest, Selenium (Page Object Model) и интегрированные с Allure для генерации отчетов.

## Предварительные требования

 **Allure CLI**: Должен быть установлен в систему.
   - *macOS*: `brew install allure`
   - *Windows*: через Scoop (`scoop install allure`) или скачав архив с GitHub и добавив в PATH.

## Установка зависимостей проекта

Установите необходимые библиотеки из корня проекта:
```bash
pip install -r requirements.txt
```
*(Убедитесь, что в файле есть `pytest`, `selenium` и `allure-pytest`)*

## Запуск тестов и генерация отчета

### Шаг 1. Запуск тестов для сбора данных
Выполните команду для запуска тестов. Флаг `--alluredir` указывает папку, куда будут складываться временные JSON-файлы с результатами.
```bash
pytest lesson_10/ --alluredir=allure-results
```

### Шаг 2. Просмотр отчета
Чтобы сгенерировать красивый HTML-отчет во временной папке и автоматически открыть его в браузере, выполните:
```bash
allure serve allure-results
```

### Шаг 3. (Опционально) Сборка статичного отчета
Если вам нужно сохранить отчет в виде папки с HTML-файлами (например, для отправки коллегам или загрузки на CI/CD):
```bash
allure generate allure-results -o allure-report --clean
```
Для открытия уже сгенерированного локального отчета из папки `allure-report`:
```bash
allure open allure-report
```
